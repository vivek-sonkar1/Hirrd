# Hirrd

A modern job recruitment platform where **recruiters** can post and manage job listings, and **candidates** can explore opportunities, submit resumes, and track their application status.  
Built with **React**, **Tailwind CSS**, **ShadCN UI**, **Supabase**, and **Clerk** for seamless authentication and role-based access control.
